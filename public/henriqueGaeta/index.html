Hi my name is Henrique and for 10 years I dedicated myself to learn and teach science and biology, one of my many passions. By doing so I was fortunate enough to work with amazing people in different universities and places.

As I have always been an ​​technology enthusiast I started working with molecular and chemistry biology using data analysis and statistics as a tool, so I was already working with Information Technology (IT) without realising that.

In 2018, while facing big challenges, chasing new dreams and having some spare time, I finally decided to start my new career in IT. So to pursue that dream, in that same year, I started to take Systems Analysis and Development course in Fatec (a technology college in the state of São Paulo, Brazil) and in the beginning of 2019 I started working in IT.

As an IT intern I had a big opportunity to work with a great team and learn a lot about SQL Server, C#, ASP.NET Core MVC, JavaScript, Html, CSS and Azure Cloud services. As a result, today I work as a self-employed junior software developer, building web applications, improving and bug fixing implemented services for the company I first started as an intern (SG - Car Rental Services) and some other business.

Therefore, I also develop a few other projects in Python, NodeJs, React and React Native for personal use and tests. In short, I am still learning a lot from my work colleagues and small online courses and I am happy with this new career and looking for new challenges in this amazing area.
